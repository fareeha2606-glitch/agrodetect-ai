AgroDetect AI is a smart agriculture web application built using Streamlit.
It helps farmers detect crop leaf diseases using images and provides
voice-based solutions and certification support.

## Features
- User Registration & Login
- Leaf Image Upload
- Disease Prediction
- Voice Assistance (Text-to-Speech)
- Solution Recommendation
- Certificate Generation (PDF)

## Tech Stack
- Python
- Streamlit
- gTTS
- ReportLab

## How to Run
```bash
pip install -r requirements.txt
streamlit run app.py
